if...else if和switch...case的区别

发表于 2016-05-15   |   分类于 javascript   |  

由于工作中遇到了一个判断条件较多的语句，就使用了switch…case语句，后来就研究一下if…else if和switch…case的区别

##一、if…else if语句
if…else if语句会遍历条件分支直到命中条件，然后停止。
示例代码：

if (a<1){
    x=1;
} else if (a>2) {
    x=2;
} else {
    x=3;
};

##二、switch…else语句

switch…else语句在进行比较的时候会生成一个跳转表来指示实际的case分支的地址，而这个跳转表的索引号与switch变量的值是相等的，所以只需要访问对应索引号的表项从而到达定位分支的目的。
示例代码：

switch(i){
    case 1: 
        alert(1);
        break;
    case 2: 
        alert(2);
        break;
    case 3: 
        alert(3);
        break;
    case 4: 
        alert(4);
        break;
    case 5: 
        alert(5);
        break;
    case 6: 
        alert(6);
        break;
    defaule:
        alert(7);
        break;
}

讲解：
1、default是如果前面的case都不符合，就执行defaule后面的语句。
