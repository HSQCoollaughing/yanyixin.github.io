javascript中if...else if和switch...case的用法和区别

发表于 2016-05-15   |   分类于 javascript   |  

由于工作中我们会遇到根据不同的条件执行不行的语句，这时就可以使用条件语句来实现。

这里来介绍一下在javascript中if…else if和switch…case的原理和区别。

一、if…else if语句

if…else if语句是当if中的语句成立时，则执行后面的代码，如果不成立则继续遍历其他的条件。

要在多组语句中选择一组来执行，可以使用if…else if嵌套语句。

语法：

if(条件1){
    条件1成立时要执行的代码
} else if(条件2){
    条件2成立时要执行的代码
} 
...
else if(条件n){
    条件n成立时要执行的代码
} else {
    条件1、2到n都不成立时要执行的代码
}

示例代码：

var a = 9;
if (a<=3){
  document.write("良好");
} else if (3 < a <= 6) {
  document.write("中等");
} else {
  document.write("优秀");
}

二、switch…else语句

Switch必须赋初始值，值与每个case值匹配，如果满足的话，就执行该case后的所有语句，并用break语句来阻止运行下一个case。如果所有的case值都不匹配，则执行default后面的语句。

switch…else语句在进行比较的时候会生成一个跳转表来指示实际的case分支的地址，而这个跳转表的索引号与switch变量的值是相等的，所以只需要访问对应索引号的表项从而到达定位分支的目的。

因为使用了查找表的方式，所以case的条件必须是一个连续的常量。

语法：

switch(常量表达式){
  case 常量1:
    执行语句1；
    break;    
  case 常量2:
    执行语句2；
    break;
  ...
  case 常量n:
    执行语句n；
    break;
  default:
    与case 常量1、case 常量2...case 常量n不同时执行的代码
}

if…else if

是逐一的比较条件，直到条件满足为止，当判断条件较少时，执行效率会很快。

switch…case

switch…case语句使用了二叉树算法，在进行比较的时候会生成一个跳转表来指示实际的case分支的地址，而这个跳转表的索引号与switch变量的值是相等的，所以只需要访问对应索引号的表项从而到达定位分支的目的。不需要逐个比较，会直接跳转到相应的case，当判断条件较多时，switch…case语句的效率会比较高。
