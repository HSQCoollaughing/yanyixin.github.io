沪江这一年

发表于 2017-08-26 | 分类于 沪江 | 阅读次数: | 本文总阅读量 次

填离职申请的时候，看着上面显示的入职日期，心里不禁感叹，时光荏苒，转眼一年就过去了。这一年，认识了热爱技术的小伙伴，认识了和蔼暖心的遥叔和干总，收获甚多，非常有必要做一个总结。

工作

很清晰的记得，是未来和遥叔面的我，也很清晰的记得那些面试题，就这样，顺利拿到了 offer，加入到了沪江这个大家庭，从此开始了我认为的真正的前端生涯。

来的前一个月就坐在小黑屋里，里面有遥叔、专家、王敏、晨雪、达孚还有明坤，专家特别好，有什么问题都会非常耐心的解答，我还记得有一次问王敏问题，他非常 6 的用原生就写出来了，对，当时满心的敬佩。

收银台项目

来沪江的第一个项目就是收银台，我写移动端的，珂珂来写 pc 端的，那时金融线刚成立不久，是未来带着我们。每个周五开周会，大家总结本周的任务， review 大家的代码，互相调侃，非常开心。

刚开始来的时候，未来带着我估时，有什么问题就找直接责任人，这样得到的信息才是最准确的。

通过这个项目，了解了当时沪江前端的基本架构，当然，很多东西当时还在探索当中，真正的业务代码开始写起来。

BI 组

中间到 BI 项目组支援了一段时间，认识了昕哥，昕哥超级好，非常耐心的给我们讲技术细节和难点，真的好佩服昕哥，因为这次支援，我学习了 Angular 2，有了跟之前不一样的编程思想，有幸在公司里分享 Angular的基本原理，也打扎实了 Angular 的基本用法。

在临走的时候，还总结了在 BI 这段时间的学习成果，后来昕哥送给了我一个鼠标，作为赠别之礼。当时真的超级感动，又自责没有帮昕哥很多忙。

极简收银台 sdk

这个项目应该是自己承担的第一个项目，对我的意义还是蛮大的，自己搭建了项目，了解了 webpack 的一些知识，还有打包的一些相关优化。从估时到推动项目进度，联调，第三方接入。刚开始的时候 bug 还是蛮多的，很多没有想到的地方，很多错误处理都要照顾到，合理的给出生命周期钩子，可以让调用方灵活的调用。不过这个项目做的不好的地方是，一些方法封装的不够好，关于这个，还是要继续努力的。现在第二个版本就要上线了，还是要对它负责的。

结算页

结算页是很重要的一个项目，干总也是交给我来负责了，这次是接入 node，前端也要上 redux，所以还是挺兴奋的，刚开始看 redux 确实挺吃力的，不过幸好程墨老师的那本《深入浅出React和Redux》讲的很细致，再搭配 git 上的 demo，还有和小伙伴的讨论，也是没多久就知道了 redux 的用法和它能给我们带来的好处。因为这次是迁 node，所以也要对接口做一些优化，我觉得比较重要的一点是，一定要弄明白每个字段的意思，这样在写的时候会有利于自己理解业务代码，而且方便之后修 bug，也会帮助后端同学了解里面的逻辑，一举三得。下周要把这个项目再优化一些，走之前不能留烂摊子，要让后面的同学好维护。

总结： 在项目中，一定要有高度责任感，如果这个就是你自己的产品，那你会怎么做，会怎么对他负责，他上线之后对你的影响。在项目责任感培养的过程中，不仅能够扩大自己知识面的宽度，还能让同事对你产生信任，信任是很珍贵的，一定要珍惜这种信任。

技术

  • 排错能力
  • 字段命名规范
  • 静态资源部署流程（和后端的联系是 webpack-assets.json 文件）
  • 对框架的熟悉和应用（React, Angular2）
  • 对基础更加的重视

格局

还没有很清楚的明白格局的概念，我觉得就是看待事物的态度，想法的起点，还有心胸。

就拿代码来说，要对自己写的每一行代码负责，你要知道，在你的每一行代码的上面，都有用户在真真切切的用他们。对代码负责，就是对自己的产品负责，就是对用户的负责。

技术发展上来说，我觉得主要是需求推动业务，业务推动技术发展，技术发展推动自我沉淀，这是一个正循环。
在对技术的学习方便，基础是真真切切特别重要的，看了一些大 v 知乎上发表的想法，学习任何东西基础都要打扎实，语言都是相通的，把基础学好了，其他的框架都是基于原生开发的，那理解起框架来，也是相对容易的事情了。

热情

在对技术的热情方面，一定要保持，写代码的时候，不仅要知其然，还要知其所以然。这样才能学的更深，帮助自己技术的成长。觉得看一下阮一峰老师的文章还是很有必要的。

做人

最近面试挺多家大公司，感触还是蛮多的，每个公司都有不同的文化，相同的是对技术的要求。所以基础真的特别重要！新美大的阿坚说：“不仅要关注新的技术，基础也要打扎实，阮一峰老师的文章在看的同时，最好亲手写一些 demo”。

所以光看是不行的，要配合实战演练，才能深刻了解其中的含义，这个我要特别重视。

每一个成功都是一步一个脚印踏出来的，成功并不会无缘无故的出现，看别人受到领导重视，那是因为人家有能力有实力，这个没有捷径。踏踏实实的把每个工作做好，不仅要做好，还要把普通的工作做出惊喜。多想想为什么，有什么更好的办法，养成习惯之后，对知识的吸收力度自然也就提高了。

末

很开心认识那么多小伙伴，学到那么多的知识，也感觉遥叔、干总和老大对我的教导，你们的话真的很受用，帮我从岔路口给拽回来，为我那么操心，真的好感动啊。
